Please follow below instructions to play the Spellings Quiz

  1. To start with, please select a quiz type among four options - 'Colors', 'Animals', 'Birds', 'Fruits'.

  3. Please note that only on screen keyboard could be used for typing.
  4. You can click on 'Try Later' button, if you don't know the spelling and want to try later.
  5. 'Backspace' button could be used if any correction is needed.
  6. You can see the score at the bottom of the page.
  7. Once the quiz is finished, a message will be displayed including final score.
  8. New quiz could be started at any time.
